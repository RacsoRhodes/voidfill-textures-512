	About

		512x512 textures for the Minetest mod "Voidfill".

	Install

		Copy or move the image files from this directory to the Minetest texture
		pack you are currently using.

		For example, if you are using "HDX 512", you would move these files to
		the textures/HDX_512px directory in your Mintest installation directory.

	Voidfill on GitHub

		https://github.com/RacsoRhodes/voidfill
